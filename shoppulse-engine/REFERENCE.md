# ShopPulse Distribution Strategy Reference

## Core insight
Do not give away a critique tool. Give away a measured decision tool.
That filters for active sellers, demonstrates unique value, and creates
a natural upgrade path.

## What competitors actually do
- EverBee: free Hobby plan + extension install, 30% affiliate (180-day cookie), blog/podcast/Facebook/YouTube/webinars
- RankHero: free public tools with daily limits, no account required
- Alura: free plan + extension + affiliate program
- eRank: Listing Changes tool tracks daily snapshots of edits vs metrics
- All: free utility first → trust via education/community → monetize deeper workflow

## What users pay for
1. Research convenience (product analytics, keywords, trends)
2. Actionable optimization (not just numbers — what to do)
3. Measurement after edits (did this change work?)
4. Human audits/coaching (service market, $10-150, not SaaS moat)

## ShopPulse Pulse Check (free entry point)
- Seller provides data for ONE listing
- ShopPulse identifies the single highest-confidence issue
- Gives one ranked action
- Defines what metric should move
- Tracks that listing for 14 days
- Labels result: helped / no clear evidence / hurt / inconclusive
- Filters for active sellers (not beginners)
- Natural upgrade to full-shop action feeds

## Distribution plan (3 steps)
1. Free tool aimed at right ICP (requires real shop context, not frictionless toy)
2. Borrow trust from Etsy educator ecosystem (small-to-mid coaches, podcasters, newsletter creators)
3. Add affiliates later after landing page converts

## Outreach structure
Subject: "I built a free Etsy Pulse Check that tells a seller the #1 listing change to make next and tracks the result for 14 days."
Body: "It's for active Etsy shops that already have enough traffic to test changes. Looking for 5-10 shops to run through it free in exchange for honest feedback."
Ask: "Would you be open to trying it on one listing or sharing it with a few sellers in your audience?"

## Content topics (aligned with paid wedge)
- "How to know if an Etsy listing change actually worked"
- "Views up, sales flat: what to fix first"
- "Favorites but no orders: what that usually means"
- "How to use Etsy Search Visibility without guessing"
- "What to change first on a listing with traffic but weak conversion"
